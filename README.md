## Setup Instructions

### Dependencies
* Clojure
* Java
* NPM

### Install JS Dependencies
```
npm install
```

### Build App CLJS
```
npm run app-cljs-dev
```

### Build Server CLJS
```
npm run server-cljs-dev
```

### Build App JS
```
npm run app-js-dev
```

### Build Server JS
```
npm run server-js-dev
```

### Clean Build
```
npm run clean
```

### Clear Build Cache
```
npm run clear-cache
```
